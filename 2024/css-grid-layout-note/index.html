這篇是給未來的自己當備忘錄的 grid 排版筆記。我流重點如下：

  • 要佔滿剩下所有的空間時，使用 fr
  • 只拿自己需要的空間時，使用 min-content
  • 需要明定尺寸範圍時，使用 minmax([保底尺寸], [最大使用尺寸])
