Summary of this post

An introduction to hexo’s category hierarchy.

Categories with hierarchy

# tag:
- Fruit
- Apple

# Result:
└── Fruit
    └── Apple

Categories without hierarchy (no hierarchy, parallel)

# tag:
- [Fruit, Apple]
- [Vegetable]

# Result:
├── Fruit
|   └── Apple
└── Vegetable

Multiple parallel categories

# tag:
- [Fruit, Apple]
- [Fruit, Orange]
- [Vegetable]

# Result:
├── Fruit
|   └── Apple
|   └── Orange
└── Vegetable

Sub-sub-category

# tag:
- [Fruit, Apple, Apple seed]
- [Vegetable]

# Result:
├── Fruit
|   └── Apple
|      └── Apple Seed
└── Vegetable
