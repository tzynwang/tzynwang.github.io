專案結構

透過一行腳本在多個資料庫個別產生種子資料

npm run腳本如右："seed": "node models/seeds/categorySeeder.js && node models/seeds/recordSeeder.js"
兩份種子腳本的原始碼如下：

  • node models/seeds/categorySeeder.js執行完畢後，需確保腳本停止，方可開始執行node models/seeds/recordSeeder.js
  • 在categorySeeder.js中，待Categories.create()完成後，串接.then()來停止db連線
  • categorySeeder.js結束後，開始執行recordSeeder.js
  • 根據MDN的forEach() polyfill，forEach()並沒有await callback function的機制，故改使用for...of來進行迴圈；相關參考文件如下：
    • 為什麽 array.foreach 不支持 async/await
    • Why does async/await in a .forEach not actually await?
    • Using async/await with a forEach loop
  • 待for...of執行完畢後，再關閉db連線，結束recordSeeder.js
