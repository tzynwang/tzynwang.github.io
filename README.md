# 普通文組 Normal Reader 2.0

## 簡介

這個 repo 是技術部落格[普通文組 2.0](https://tzynwang.github.io/) 的本體，由查理（tzyn.wang🦊gmail.com）維護。

## 快速啟動

```bash
make i
make start
```

## 基礎設施

- Node.js: v20
- Package Manager: `yarn`
